# YouTube Clone
A basic YouTube clone with features like history, liked videos, watch later, and video upload.

## Technologies Used
Frontend: ReactJS <br>
Backend: NodeJS <br>
Database: MongoDB <br>
Deployment <br>
Frontend: Netlify Link <br>
Backend: Hosted on Vercel <br>
Installation <br>

## Clone the repository:

## Backend:
cd backend<br>
npm install <br>
npm start <br>
## Frontend:
cd frontend <br>
npm install <br>
npm start <br>
